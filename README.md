## ✨ Features

- 📅 Add, edit, and delete daily expenses  
- 🧾 Categorize expenses (food, travel, bills, etc.)  
- 📊 View total spending at a glance  
- 🔍 Filter expenses by date or category  
- 💾 Data stored safely for future reference

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```bash
git clone https://github.com/sahilbehera1012007-cyber/2nd-Sem-Project---Expense-Tracker.git
cd 2nd-Sem-Project---Expense-Tracker
```

### 2️⃣ Install dependencies

If this is a simple HTML/CSS/JS project, you can skip this step.  
If you are using a backend (Node, etc.), mention:

```bash
npm install
```

### 3️⃣ Run the project

- If it is static:  
  - Open `index.html` in your browser  
- If using Node or any backend server:  

```bash
npm start
# or
node index.js
```
